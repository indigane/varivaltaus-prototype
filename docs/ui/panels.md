# UI panels

Use plain HTML controls.

## Minimum panels

### Game setup

- board type,
- board size,
- color count,
- player count,
- bot/human per player,
- team assignment,
- capture mode,
- win condition,
- start/reset button.

### Current game

- current player,
- color buttons,
- scores,
- turn number,
- winner/game status.

### Debug

- save JSON,
- load JSON,
- replay moves,
- validate graph,
- bot autoplay toggle.

## Prototype UI priority

Make it functional first.

Do not spend much time on visual polish until the rules are fun.
