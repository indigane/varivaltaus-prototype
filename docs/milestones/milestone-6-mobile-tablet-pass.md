# Milestone 6: mobile and tablet playtest pass

## Goal

Make the prototype practical for playtesting on phones and tablets.

## Build

- responsive layout,
- larger color buttons,
- canvas scaling,
- pointer/touch input,
- mobile-friendly setup flow,
- smaller default board presets.

## Done when

- game is playable on a phone browser,
- game is playable on a tablet browser,
- no hover-only controls are required,
- color buttons are easy to tap,
- board remains readable.
