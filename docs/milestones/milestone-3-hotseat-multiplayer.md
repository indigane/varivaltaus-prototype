# Milestone 3: hot-seat multiplayer

## Goal

Support any number of local players.

## Build

- player setup UI,
- ownership initialization,
- turn order,
- scores,
- win detection,
- player display.

## Done when

- 2-player duel works,
- 3+ player games work,
- scores are correct,
- current player advances correctly,
- game ends correctly.
