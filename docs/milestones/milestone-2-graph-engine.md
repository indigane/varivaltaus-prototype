# Milestone 2: graph engine

## Goal

Ensure the prototype is not square-grid-specific.

## Build

- `core/graph.js`,
- board validation,
- square generator outputs graph,
- renderer draws polygons only,
- move algorithm uses neighbors only.

## Done when

- game rules never use row/column logic,
- board validation catches invalid graphs,
- square board still works through generic graph path.
