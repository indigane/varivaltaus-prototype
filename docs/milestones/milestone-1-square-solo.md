# Milestone 1: playable square solo game

## Goal

Create the simplest playable local game.

## Build

- `index.html`
- `style.css`
- `js/main.js`
- square board generator,
- Canvas renderer,
- palette,
- one-player flood move,
- move counter,
- reset button.

## Rules

- one player,
- neutral-only capture,
- fixed turn limit or target percent,
- square board only.

## Done when

- user can start a game,
- user can choose colors,
- territory expands correctly,
- score updates,
- game can end,
- reset works.
