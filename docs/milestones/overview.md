## Prototype done criteria

The prototype is successful when playtests can answer:

- Is solo play fun?
- Is 1v1 fun?
- Is neutral-only capture better than enemy capture?
- Are 3–6 player free-for-alls fun or chaotic?
- Are teams fun?
- Which tilings are strategically interesting?
- How many colors feel good?
- Are bots useful as opponents?
- Which board sizes work on phones/tablets?
- Which rules should be removed before production?
