# Milestone 5: multiple tilings

## Goal

Validate whether non-square boards are fun.

## Build

- triangle board,
- hex board,
- custom JSON board loader,
- later Cairo pentagon board,
- optional masks.

## Done when

- same game rules work on square, triangle, and hex boards,
- renderer does not need tiling-specific code,
- graph validation passes for all generators,
- custom map loader can load a small hand-authored map.
